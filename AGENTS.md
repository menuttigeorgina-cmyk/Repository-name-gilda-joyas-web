# Proyecto GIVEM Joyas

Este proyecto es una tienda ecommerce headless para GIVEM Joyas.

## Objetivo
Crear una demo funcional para una entrega académica de IA No Code:
- Catálogo conectado a WooCommerce.
- Frontend en Next.js.
- Carrito funcional.
- Checkout simulado dentro de la web.
- Chatbot básico para consultas frecuentes.
- Diseño elegante, cálido, premium e inclusivo.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- WooCommerce REST API
- WordPress como backend
- Variables privadas en `.env.local`

## Reglas importantes
- No exponer claves de WooCommerce.
- No subir `.env.local`.
- No usar WhatsApp como checkout principal.
- WhatsApp solo puede aparecer como canal de consulta.
- Mantener el diseño limpio, visual, responsive y alineado a marca de joyería.
- Evitar textos genéricos tipo lorem ipsum.
- El chatbot actual puede ser lógico/predefinido; IA generativa real queda como mejora futura.

## Marca
GIVEM Joyas es una marca de joyas moderna, delicada y versátil. No debe comunicarse como exclusiva para mujeres. El tono debe ser cálido, claro, elegante y profesional.
